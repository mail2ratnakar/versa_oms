# Privacy and Masking Note

This seed pack uses demo-only data.

Rules:

- All emails use `example.test`.
- Parent phone numbers are masked.
- Payment references are masked.
- Provider payloads are not included.
- Raw OMR files are not included.
- Answer keys are not exposed.
- Certificate files are represented as private-file metadata only.
- Signed URLs are not included.
- Public verification data is minimal.

Do not replace this pack with real student data in development.
