# Versa Seed and Mock Data Pack

This pack provides realistic demo data for the Versa Olympiads app skeleton.

## Included

- staff users and roles.
- school users.
- schools and contacts.
- exam cycles and slots.
- student roster batches.
- students and candidate IDs.
- invoices and payments.
- exam materials.
- courier shipments.
- evaluation batches and scores.
- result batches and results.
- certificates.
- notifications.
- support tickets.
- tasks.
- reports/exports.
- admin settings.
- security events and incidents.
- E2E scenario fixtures.
- load order.
- privacy and masking note.

## Purpose

Use this data for:

- UI development.
- smoke tests.
- E2E tests.
- permission tests.
- finance gate tests.
- result/certificate tests.
- export/security tests.
