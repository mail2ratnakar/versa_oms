# Role-Aware Navigation

## Rules

- Navigation is not authorization.
- Hide unavailable modules for cleaner UX.
- Always recheck permission on server.
- Staff sees modules based on role/scope.
- School sees only school portal routes.
- Public sees only verification routes.

## Staff Navigation Groups

- Dashboard.
- Schools.
- Students.
- Finance.
- Exams.
- Courier.
- Evaluation.
- Results.
- Certificates.
- Notifications.
- Support.
- Tasks.
- Reports.
- Admin.
- Security Audit.

## School Navigation

- Dashboard.
- Students.
- Payments.
- Exam Slots.
- Materials.
- Results.
- Certificates.
- Support.
- Reports.

## Mobile Navigation

- School: bottom nav or compact drawer.
- Staff: collapsible drawer.
