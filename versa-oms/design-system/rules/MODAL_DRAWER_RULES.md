# Modal and Drawer Rules

## Modal Use

Use modal only for:

- confirmation.
- simple approval/rejection.
- short reason capture.
- small warnings.

Do not use modal for:

- long forms.
- complex material release.
- result publication review.
- certificate revocation details.
- sensitive export review.

## Drawer Use

Use drawer for:

- quick preview.
- safe summary.
- row detail preview.
- audit timeline preview.
- linked record preview.

## Full Page Use

Use full page for:

- onboarding review.
- roster import.
- finance reconciliation.
- material release.
- evaluation approval.
- result publication.
- certificate revocation.
- sensitive export review.
- security incident investigation.
