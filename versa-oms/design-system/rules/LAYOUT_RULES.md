# Layout Rules

## Staff Portal

- Persistent sidebar width: 280px.
- Topbar optional; use only for environment/user/search.
- Main content padding: 32px desktop, 24px tablet, 16px mobile.
- Page header required on every module page.
- Use two-level navigation only; avoid deep nested nav.
- Important alerts appear below page header.

## School Portal

- Simpler navigation than staff.
- Use plain language.
- Primary actions must be obvious.
- Mobile-first for school users.
- Avoid exposing staff/internal terminology.

## Content Width

- Dashboard: full content width.
- Forms: max 960px.
- Detail pages: 2-column layout desktop, single-column mobile.
- Tables: full width with horizontal scroll only when necessary.

## Page Structure

Every page should follow:

1. Breadcrumbs if nested.
2. Page title and short description.
3. Primary action or status.
4. Alerts/gates.
5. Main content.
6. Audit/timeline if detail page.
