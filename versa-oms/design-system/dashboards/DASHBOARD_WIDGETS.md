# Dashboard Widgets

## Widget Rules

- Use safe summaries only.
- Never expose raw PII, OMR, answer keys or provider payloads.
- Every widget has source module.
- Drilldown rechecks source-module permission.
- Alert dismissal does not resolve source issue.
