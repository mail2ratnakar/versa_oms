# Founder Brain Checklist

Before coding:

- [ ] Brain files read.
- [ ] Skills files read.
- [ ] Latest completed/pending summary (reports/BUILD_STATUS.md) read.
- [ ] Relevant specs read.
- [ ] Relevant ADRs read.
- [ ] Relevant threat model read.
- [ ] Relevant effect chain read.
- [ ] Relevant screen contract read.
- [ ] Relevant journey test read.

Before claiming completion:

- [ ] Screen works.
- [ ] API works.
- [ ] DB persists.
- [ ] Status changes.
- [ ] Downstream effect happens.
- [ ] Audit written.
- [ ] Security/privacy checks pass.
- [ ] Journey test passes.
- [ ] CI/build/test evidence captured.
- [ ] Summary updated.
