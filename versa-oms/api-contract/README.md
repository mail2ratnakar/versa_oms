# Versa API Contract Bundle

This pack fills the missing consolidated API contract layer.

## Includes

- `openapi.yaml`
- `openapi.json`
- `API_NAMING_RULES.md`
- `REQUEST_RESPONSE_SCHEMAS.json`
- `STANDARD_ERROR_FORMAT.md`
- `IDEMPOTENCY_HEADER_CONVENTION.md`
- `PAGINATION_CONVENTION.md`
- `FILTERING_CONVENTION.md`
- `AUDIT_METADATA_CONVENTION.md`
- `SIGNED_URL_RESPONSE_CONVENTION.md`
- `PUBLIC_PRIVATE_ROUTE_ALLOWLIST.json`

## Purpose

Use this pack before frontend/backend implementation so the LLM generates consistent APIs, envelopes, errors, route names, pagination, filters, audit metadata and signed URL responses.
